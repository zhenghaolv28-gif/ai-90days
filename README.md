# AI 应用开发 90 天学习计划

## 我的背景

我是数据科学与大数据技术专业的应届本科毕业生。
目前有少量使用 AI 编写代码和修复 Bug 的经验，但 Python 和软件开发基础仍需加强。
希望1～3个月内找到AI应用相关工作

## 90 天目标

通过 90 天项目制学习，我希望：

1. 掌握够用的 Python、Git、数据库和 API 基础。
2. 能借助 AI 编写、阅读、修改和调试代码。
3. 能独立完成并部署一个 AI 网站或小应用。
4. 能搭建自动化工作流。
5. 完成一个求职辅助 Agent。
6. 建立可以用于求职的 GitHub 作品集。

## 学习时间

- 每天投入：2～5 小时
- 计划周期：90 天
- 学习方式：任务驱动、项目实践、每日复盘

## 计划完成的项目

### 项目一：业务任务与资料管理台

实现任务管理、搜索筛选、数据保存和 CSV 导入导出。

### 项目二：岗位—简历智能匹配器

分析岗位要求与个人简历，输出匹配度、能力缺口和简历修改建议。

### 项目三：JobFlow 求职 Agent

实现岗位整理、匹配分析、简历修改建议、材料生成和人工确认流程。

## 我的训练规则

1. 每天都要产生可运行的结果。
2. AI 生成的核心代码必须能够解释。
3. 遇到 Bug 时，先记录现象、报错和复现步骤。
4. AI 说“已经修复”不算完成，必须亲自运行验证。
5. 每完成一个小阶段就提交一次 Git。
6. 不把密码、API 密钥和个人隐私提交到仓库。
7. 每天记录完成内容、问题和复盘。
